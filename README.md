###### AssemblyStuff

## [Here](https://github.com/Quesmin/AssemblyStuff/blob/master/macro.asm) are some macros for input/output such as:
  * input/output a string
  * input/output a number (also signed number)
  * input/output an array of numbers
  * input/output a char
  * print "INPUT: "
  * print "OUTPUT: "
  * print a space
  * print endline
